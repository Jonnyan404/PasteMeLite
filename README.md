  <h1>PasteMe Lite</h1>
</div>

[PasteMe](https://github.com/LucienShui/PasteMe) 的轻量版本，将前端集成至 Golang 服务中，不依赖于数据库服务。唯一一件需要做的事情，就是运行。

## Deploy

`git submodule update --init --recursive`
